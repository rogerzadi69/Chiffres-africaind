# Compter en chiffres africains

Parcours d'alphabétisation numérique en quatre modules, fondé sur un système
de numération africain à logique constructive.

## Les modules

| Étape | Fichier | Description |
|-------|---------|-------------|
| Accueil | `index.html` | Portail du parcours |
| 1 | `apprendre-a-tracer.html` | Tracer les chiffres africains (jeu d'écriture au doigt) |
| 2 | `lire-les-chiffres.html` | Quiz de lecture : des glyphes africains aux chiffres arabes |
| 3 | `ecrire-les-chiffres-arabes.html` | Tracer les chiffres arabes, guidé par le glyphe africain |
| 4 | `calculatrice-africaine.html` | Calculatrice à double écran (africain + conversion arabe) |

## Utilisation

Aucune installation : chaque page est autonome (HTML/CSS/JS, images intégrées).
Ouvrir `index.html` dans un navigateur. Fonctionne hors ligne.

## Déploiement

Site 100 % statique : compatible GitHub Pages, Vercel, Netlify, ou simple clé USB.
